3. Skúška 25.6.2026¶

Poznámka

  • riešenia úloh ulož do jedného súboru a odovzdaj na testovací server https://vektor.fmph.uniba.sk/

    • prvé tri riadky súboru s riešením musia obsahovať:

      # skuska variant C
# student: Janko Hraško
# datum: 25.6.2026

      zrejme ako študenta uvedieš svoje meno.

  • nepoužívaj global ani import

  • nemeň mená už zadaných atribútov (metód a premenných)

  • do existujúcej triedy môžeš pridávať vlastné atribúty a metódy (okrem typu list, tuple a dict)

  • pri testovaní riešenia sa bude kontrolovať aj štruktúra vytvoreného grafu

  • môžeš si stiahnuť 2sk_3.zip s kostrou programu a testovacími súbormi 'subor1.txt', 'subor2.txt', 'subor3.txt', …, ktoré používa testovač.


Podmorská výskumná stanica sa skladá z viacerých modulov, ktoré sú nejako pospájané prechodovými tunelmi. Každý takýto modul má zásoby kyslíka a potravín len na niekoľko dní prežitia a preto si výskumníci musia dobre zvážiť, v akom poradí budú tieto moduly obývať. Predpokladáme, že každý modul, v ktorom výskumníci bývali, sa po ich odchode automaticky uzamkne a nedá sa už do neho znovu vojsť - toto platí pre všetky moduly okrem štartového, ktorý bude slúžiť ako únikový pre ukončenie celej expedície a cez neho sa dostanú opäť na povrch.

Úlohou bude pre zadaný štartový modul navrhnúť také poradie modulov, v ktorých budú postupne výskumníci prebývať, ktoré im zabezpečí čo najdlhšie trvajúcu podmorskú expedíciu. Zrejme prvý a posledný modul v tomto poradí bude rovnaký a teda štartový.

V súbore máme informácie o moduloch a prechodových tuneloch. Meno každého modulu začína jedným alebo viac písmenami a je ukončené celým číslom, ktoré pre tento modul vyjadruje počet dní na prežitie. Napríklad, abc42 označuje meno modulu, v ktorom sa dá prežiť 42 dní. Všetky moduly majú rôzne mená. V súbore sú informácie v tvare dvojíc mien modulov, napríklad:

abc42-abc11
xy37-abc11
a42-abc42

Zadefinuj metódy triedy PodmorskaStanica:

class PodmorskaStanica:
    def __init__(self, meno_suboru):
        self.mena = []
        self.tab = []
        ...

    def start(self, modul):
        ...
        return ...

    def zablokuj(self, *moduly):
        ...
        return ...

    def __getitem__(self, modul):
        ...
        return ...

Metódy:

  • __init__(meno_suboru) prečíta zadaný textový súbor a vytvorí reprezentáciu neorientovaného neohodnoteného grafu pomocou tabuľky susedností (atribút tab bude dvojrozmerný zoznam hodnôt True a False), pričom poradie vrcholov v tomto grafe bude uložené v atribúte mena (typu list); okrem týchto dvoch atribútov nepoužívaj žiadne ďalšie štruktúrované atribúty (typu list, tuple, dict) - povolené sú len pomocné atribúty typu množina, resp. lokálne premenné ľubovoľných typov

  • start(modul) nájde cyklus v grafe (začína aj končí v zadanom module), ktorý umožní čo najdlhšie prebývanie v moduloch (maximálny súčet dní)

    • metóda vráti dvojicu: počet dní na prebývanie v moduloch a cyklus ako n-tica mien modulov (tuple)

    • ak je takýchto cyklov viac, metóda vráti ľubovoľný z nich

    • ak bol štartový modul zrušený, metóda vráti dvojicu (0, ())

    • ak je v grafe tento štartový vrchol izolovaný, metóda okrem počtu dní vráti (modul,)

    • napríklad, aj takýto cyklus môže byť korektným riešením: (modul, modul1, modul)

  • zablokuj(*moduly) nejako si označí vrcholy, ktoré by sa nemali používať, lebo sa zatopili vodou

    • toto zablokovanie bude fungovať len pre najbližšie volanie metódy start() a hneď potom sa tieto moduly odblokujú

    • metóda vráti počet nezablokovaných modulov

  • __getitem__ vráti číselnú informáciu o počte dní pre tento modul

    • napríklad, pre abc42 vráti celé číslo 42

Testovač môže spúšťať metódu start() viackrát s rôznymi štartovými modulmi, pripadne s rôzne zablokovanými modulmi.

Napríklad, pre takéto zadanie grafu:

a12-b15
b15-d18
c23-b15
d18-c23
d18-e15
e15-a12

takýto test:

if __name__ == '__main__':
    st = PodmorskaStanica('subor1.txt')
    print(f'{st.mena = }')
    print(f'{st.tab[st.mena.index('e15')][st.mena.index('d18')] = }')
    print(f'{st.tab[st.mena.index('e15')][st.mena.index('c23')] = }')
    print(f'{st['c23'] = }')
    print(f'{st.start('a12') = }')
    print(f'{st.zablokuj('c23') = }; {st.start('a12') = }')
    print(f'{st.zablokuj('c23') = }; {st.start('d18') = }')
    print(f'{st.zablokuj('a12') = }; {st.start('d18') = }')
    print(f'{st.zablokuj('b15') = }; {st.start('d18') = }')
    print(f'{st.zablokuj('b15', 'e15') = }; {st.start('a12') = }')
    print(f'{st.zablokuj('d18') = }; {st.start('d18') = }')

vypíše, napríklad:

st.mena = ['d18', 'c23', 'b15', 'e15', 'a12']
st.tab[st.mena.index('e15')][st.mena.index('d18')] = True
st.tab[st.mena.index('e15')][st.mena.index('c23')] = False
st['c23'] = 23
st.start('a12') = (83, ('a12', 'e15', 'd18', 'c23', 'b15', 'a12'))
st.zablokuj('c23') = 4; st.start('a12') = (60, ('a12', 'b15', 'd18', 'e15', 'a12'))
st.zablokuj('c23') = 4; st.start('d18') = (60, ('d18', 'e15', 'a12', 'b15', 'd18'))
st.zablokuj('a12') = 4; st.start('d18') = (56, ('d18', 'b15', 'c23', 'd18'))
st.zablokuj('b15') = 4; st.start('d18') = (41, ('d18', 'c23', 'd18'))
st.zablokuj('b15', 'e15') = 3; st.start('a12') = (12, ('a12',))
st.zablokuj('d18') = 4; st.start('d18') = (0, ())
