2. Priebežný test - variant B¶

Poznámka

  • riešenia úloh ulož do jedného súboru a odovzdaj na testovací server https://vektor.fmph.uniba.sk/

    • prvé tri riadky súboru s riešením musia obsahovať:

      # 2. test variant B
# student: Janko Hrasko
# datum: 15.12.2025

      zrejme ako študenta uvedieš svoje meno.

  • nepoužívaj global ani import

  • svojim funkciám (oproti zadaniu) môžeš zmeniť mená parametrov, môžeš si zadefinovať pomocné funkcie


  1. Prvé štyri riadky Pascalovho trojuholnika zvykneme vypisovať takto:

          1
    1   1
  1   2   1
1   3   3   1

    Napíš funkciu riadok(n), ktorá vráti n -tý riadok (číslujeme od 0) tohto trojuholníka. Napríklad:

    >>> riadok(3)
    [1, 3, 3, 1]

  1. V textovom súbore sú slová oddelené medzerami, prípadne koncami riadkov. Napíš funkciu vyber(meno_suboru), ktorá prečíta takýto súbor a vráti n-ticu (tuple) všetkých čísel a logických konštánt True a False. Do n-tice ich zaraď presne v tom poradí, ako sú v súbore. Napríklad, ak by súbor obsahoval iba:

    abc 3x 4.7 (75) == True 8

    funkcia by vrátila:

    (4.7, True, 8)

  1. Dodefinuj metódy triedy:

    class Cislo:
    ...

    tak, aby fungovalo:

    >>> i = Cislo(456)     # parametrom je kladné číslo
>>> i
    456
>>> i.daj()            # pôvodná hodnota
    456
>>> i.daj(0)           # nultá cifra od konca
    6
>>> i.daj(2)           # druhá cifra od konca
    4
>>> i.daj(5)           # vráti None
>>> i.daj(0)
    6
>>> i.dalsi()          # predchádzajúca cifra
    5
>>> i.dalsi()          # predchádzajúca cifra
    4
>>> i.dalsi()          # vráti None

  1. Vygeneruj všetky výrazy s tromi celočíselnými operandami v tvare 'a # b # c', kde a, b, c sú zo zadanej postupnosti hodnôt a operácie # sú buď + alebo -, napríklad '3 + 2 - 1' alebo '7 - 2 - 3'. Napíš funkciu vypocty(*hodnoty), ktorá vráti množinu reťazcov: každý reťazec obsahuje výraz, ale bez medzier, ku torému je prilepené '=' a hodnota tohto výrazu, napríklad v tvare '3+2-1=4'. Ak postupnosť hodnôt obsahuje len jednu hodnotu, výsledok bude takýto:

    >>> vypocty(3)
    {'3-3-3=-3', '3+3-3=3', '3+3+3=9', '3-3+3=3'}

  1. Napíš funkciu pocitaj(*post) s nepárnym počtom parametrov: každý parameter na nepárnom mieste je celé číslo, každý párny parameter je celočíselná funkcia (jej referencia) definovaná s dvoma parametrami. Funkcia postupne vyhodnotí takto zadanú postupnosť: postupne zoberie najprv prvé tri hodnoty (teda číslo, funkcia, číslo), vypočíta ich hodnotu (volaním funkcia(číslo, číslo)) a ďalej pokračuje s touto vypočítanou hodnotou a ďalšími dvoma parametrami, … Napríklad:

    >>> pocitaj(7)
    7
>>> f = lambda a, b: 3*a-2*b
>>> pocitaj(7, f, 6)
    9
>>> pocitaj(7, f, 6, int.__add__, 5)
    14

  1. Zadefinuj takúto mapovaciu funkciu map_2(funkcia, postupnost), ktorá aplikuje danú funkciu (prvý parameter) na všetky prvky nejakej postupnosti a z výsledkov poskladá nový zoznam. Ak je niektorý prvok v postupnosti zoznam alebo n-tica, aplikuje toto mapovanie aj na jeho prvky. Zrejme aj tieto podzoznamy môžu obsahovať ďalšie podzoznamy, na ktoré bude treba aplikovať danú funkciu. Napríklad:

    >>> map_2(int, [[[('37',)]]])
    [[[[37]]]]

  1. Preprogramuj:

    def rob(x, y, n, d):
    if n > 0:
        canvas.create_line(x, y, x, y - d)
        rob(x, y - d, n - 1, d * .5)
        if n % 2:
            canvas.create_line(x, y, x - d, y - d)
            rob(x - d, y - d, n - 1, d * .6)
        else:
            canvas.create_line(x, y, x + d, y - d)
            rob(x + d, y - d, n - 1, d * .7)
    else:
        canvas.create_oval(x - 2, y - 2, x + 2, y + 2)

    tak, aby funkcia rob, ale bez kreslenia, vygenerovala (pomocou yield) postupnosť súradníc (teda tuple dvoch čísel) stredov malých kružníc. Napríklad, volanie:

    li = [*rob(200, 250, 5, 100)]

    by malo vrátiť zoznam dvojíc všetkých súradníc v správnom poradí.
