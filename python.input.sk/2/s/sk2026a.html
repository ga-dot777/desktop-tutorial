1. Skúška 9.6.2026¶

Poznámka

  • riešenia úloh ulož do jedného súboru a odovzdaj na testovací server https://vektor.fmph.uniba.sk/

    • prvé tri riadky súboru s riešením musia obsahovať:

      # skuska variant A
# student: Janko Hraško
# datum: 9.6.2026

      zrejme ako študenta uvedieš svoje meno.

  • nepoužívaj global ani import

  • môžeš definovať ďalšie pomocné metódy, okrem vertices nedefinuj ďalšie atribúty

  • môžeš si stiahnuť 2sk_1.zip s kostrou programu a testovacími súbormi 'subor1.txt', 'subor2.txt', 'subor3.txt', …, ktoré používa testovač.


Dopíš metódy triedy Graph:

inf = float('inf')

    class Vertex:
        def __init__(self, x, y):

        def __repr__(self):
            return f'Vertex{self.xy}'

        self.vertices = []                         # jednorozmerné pole vrcholov typu Vertex
        ...

        ...

    def get_weight(self, vertex1, vertex2):        # vráti váhu hrany
        ...

    def get_distance(self, vertex1, vertex2):      # vráti najkratšiu vzdialenosť (súčet váh)
        ...

    def get_path(self, vertex1, vertex2):          # vráti postupnosť vrcholov z vertex1 do vertex2
        ...

Pomocou metód triedy Graph vieme nájsť najkratšie cesty (súčet váh) v ohodnotenom neorientovanom grafe pomocou Dijkstrovho algoritmu. Tento graf reprezentuje vrcholy, ktoré sú bodmi v rovine, pričom hranami môžu byť spojené ľubovoľné dva vrcholy. Váhou hrany je potom druhá mocnina geometrickej vzdialenosti dvoch bodov v rovine. Súradnice bodov, ktoré reprezentujú vrcholy grafu sú zadané v textovom súbore: v každom riadku sú dve celé čísla (x, y).

Inicializácia triedy Graph dostáva meno textového súboru s popisom vrcholov grafu a vytvorí z neho zoznam vrcholov (do atribútu vertices). Metódy get_distance a get_path využijú výsledok Dijkstrovho algoritmu - snažte sa tento algoritmus nevolať zbytočne, ak sa dá výsledok zistiť z posledného volania algoritmu. Tento algoritmus namiesto poľa D (s vypočítanými najkratšími dĺžkami) využije atribút d v každom vrchole. Metóda get_path vráti postupnosť vrcholov (zoznam list s prvkami typy Vertex) pričom prvým prvkom postupnosti je vertex1 a posledným vertex2. Volanie get_path(vertex1, vertex1) vráti postupnosť dĺžky 1. Metóda get_weight vráti váhu hrany, t.j. druhú mocninu vdialenosti dvoch bodov v rovine.

Podtriedu Vertex nemá zmysel meniť, lebo testovač použije pôvodnú verziu.

Testovať program môžeš, napríklad takto:

>>> g = Graph('subor1.txt')
>>> v1 = g.get_vertex(66, 5)
>>> v2 = g.get_vertex(68, 93)
>>> g.get_weight(v1, v2)
7748
>>> g.get_distance(v1, v2)
5256
>>> g.get_path(v1, v2)
[Vertex(66, 5), Vertex(99, 43), Vertex(99, 52), Vertex(68, 93)]
